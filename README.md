# XYZ E-Commerce Sales Dashboard

## 📊 Project Overview
This Power BI dashboard analyzes XYZ E-Commerce sales data and provides insights into:
- Sales Performance
- Profit Analysis
- Quantity Sold
- Average Order Value (AOV)
- State-wise Sales
- Category & Sub-Category Analysis
- Monthly Profit Trends
- Payment Mode Analysis

---

## 🚀 Tools & Technologies Used
- Power BI
- Excel / CSV Dataset
- Data Visualization
- Data Cleaning

---

## 📌 Dashboard Features
- Interactive Filters & Slicers
- KPI Cards
- Dynamic Charts
- Monthly Profit Analysis
- Category-wise Quantity Analysis
- State-wise Sales Insights
- Payment Mode Distribution

## 📈 Key Insights
- Maharashtra generated the highest sales amount.
- Clothing category contributed the highest quantity sold.
- COD was the most preferred payment mode.
- Profit varied significantly across months.
